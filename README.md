# Програма для планування асинхронних задач

Ця програма демонструє використання планувальника задач у Spring Boot для реалізації двох основних завдань:

1. **Задача 1**: Лічильник, який збільшується на 1 кожні 2 секунди, зупиняється після досягнення значення 10.
   - Кожні 2 секунди лічильник збільшується на одиницю.
   - Після досягнення значення 10 виводиться повідомлення про завершення.

2. **Задача 2**: Виведення поточного часу кожні 5 секунд від запуску програми, зупиняється після виконання умови зупинки лічильника (коли лічильник досягне 10).

## Структура проекту

- **CounterService**: Сервіс, що займається лічильником і його інкрементуванням.
- **TimeService**: Сервіс, що виводить поточний час кожні 5 секунд.
- **Application**: Головний клас для запуску програми.

## Вимоги

- Java 17 або новіша.
- Spring Boot 3.x.

## Інструкція по запуску

### 1. Клонуйте репозиторій:


### 2. Інсталяція залежностей:

Використовуйте Gradle для завантаження необхідних залежностей:

```bash
./gradlew build
```

### 3. Запуск програми:

Запустіть програму за допомогою:

```bash
./gradlew bootRun
```

### 4. Перевірка роботи:

- Програма виводитиме значення лічильника кожні 2 секунди.
- Після досягнення значення 10 лічильник зупиняється, і виводиться повідомлення.
- Кожні 5 секунд програма буде виводити поточний час.
- Програма зупиняє виведення поточного часу після досягнення лічильником значення 10.

## Опис коду

### `CounterService.java`

Цей сервіс відповідає за інкрементування лічильника:

- Метод `incrementCounter()` збільшує значення лічильника на 1 кожні 2 секунди.
- Коли лічильник досягає значення 10, зупиняється подальше інкрементування.

### `TimeService.java`

Цей сервіс виводить поточний час кожні 5 секунд:

- Метод `printCurrentTime()` виводить поточний час, поки лічильник не досягне 10.

### `Application.java`

Головний клас, який запускає програму:

- Запускає лічильник та планувальник часу.
- Використовує автоматичний запуск методів, помічених анотацією `@Scheduled`.
